## 目标

在 MS-002（设备监控上线）稳定交付后，根据用户真实 Azure 环境（A100 `mux-a100`，动态公网 IP）对齐面板可用性，并收尾此前列为"后期"的增强能力。本里程碑聚焦三条主线：

1. **REQ-002 真实对齐**：让面板对真实 A100 真正可用——只读 Service Principal（`Reader`）认证落地，A100 动态公网 IP 经 Azure 自动解析后用于 SSH GPU 采集，非 running 时跳采。
2. **GPU 趋势 + AI 额度**：完成 GPU 历史降采样/趋势查询与前端迷你图（REQ-002 增强）、AI 使用额度监控端到端（REQ-004）。

本里程碑完成后，用户打开面板即可看到真实 A100 的电源态/GPU 实时与趋势、AI 各服务的额度，且系统可在树莓派上长期稳定运行而不耗尽磁盘。
